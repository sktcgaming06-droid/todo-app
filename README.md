# 📝 To-Do List App

A beautiful, modern to-do list application built with HTML, CSS, and JavaScript.

## Features

✅ **Add Tasks** - Quickly add new tasks to your list
✅ **Mark Complete** - Click any task to mark it as done
✅ **Delete Tasks** - Remove tasks you no longer need
✅ **Auto-Save** - Tasks are saved automatically in your browser
✅ **Responsive Design** - Works perfectly on desktop and mobile

## How to Use

1. Open `index.html` in your web browser
2. Type a task in the input box
3. Click "Add Task" or press Enter
4. Click a task to mark it complete
5. Click "Delete" to remove a task

## Files

- `index.html` - Main page structure
- `style.css` - Beautiful styling
- `script.js` - Functionality and logic

## Technologies

- HTML5
- CSS3
- JavaScript (Vanilla)
- LocalStorage API
